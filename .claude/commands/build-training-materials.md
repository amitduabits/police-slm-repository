Create complete training materials for Gujarat Police officers to use the AI system.

## Training Curriculum (5 modules)

### MODULE 1: Introduction (30 min) - `docs/training/module1_introduction.md`
- What is AI and how does it help investigations? (simple language)
- What this system CAN do (suggest steps, review chargesheets, find similar cases)
- What this system CANNOT do (replace officer judgment, guarantee accuracy)
- Privacy: how data is protected (air-gapped, encrypted, audited)
- Quick demo walkthrough with screenshots

### MODULE 2: SOP Assistant (45 min) - `docs/training/module2_sop.md`
- How to enter FIR details
- Understanding the output (priority levels, source references)
- When to follow vs deviate from suggestions
- 3 practice exercises with different FIR scenarios

### MODULE 3: Chargesheet Reviewer (45 min) - `docs/training/module3_chargesheet.md`
- How to upload a draft chargesheet
- Reading the completeness report (red/yellow/green flags)
- Using reference chargesheets panel
- 2 practice exercises

### MODULE 4: Case Search (30 min) - `docs/training/module4_search.md`
- Writing effective search queries
- Using filters (date, district, category, section)
- Finding similar cases
- 5 practice scenarios

### MODULE 5: Pattern Insights (20 min) - `docs/training/module5_analytics.md`
- Reading hotspot maps
- Understanding trend charts
- Monthly report generation

## For Each Module Create:
1. Full content in Markdown (English + key terms in Gujarati)
2. Quick reference card (single A5 page, printable)
3. 10-question assessment quiz with answers
4. Practice scenarios with expected outputs
